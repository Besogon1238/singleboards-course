# Необходимое аппаратное и программное обеспечение к лаораторным

## Лабораторная №1

Аппаратная часть для 1 студента:

- Одноплатник Lichee RV Dock
- 3 перемычки "мама - мама"
- UART-переходник
- Блок питания + шнур type-c для питания одноплатника


Программная часть:

- Учетка с правами администратора и sudo -i
- Установленная утилита tio
- Базовый редактор nano + vim/neovim в случае пожелания со стороны студента
