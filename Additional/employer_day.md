## Задачка на прикольный мерч

Осуществить минимальной изменения в конфигурации ядра и собрать ядро с поддержкой интерфейса I2C. Без нужных изменений поработать с экранчиком не выйдет!

### Шаг 0. Попадание в директорию сборки

Убедиться, что вы находитесь в директории сборки ядра 6.16

```
pwd
```

В выводе должно быть что то в духе:

```
блабла/бебебе/kernel-source 6.16
```

### Шаг 1. Освоение процесса конфигурации ядра

**Перед началом работы в файл конфигурации .config уже внесены множественные изменения стандартных настроек. Вам остается совершить лишь финальную донастройку**

```
make ARCH=riscv CROSS_COMPILE=riscv64-linux-gnu- menuconfig
```
Эта команда готовит исходный код ядра Linux к сборке для архитектуры RISC-V, используя кросс-компиляцию.

**После вызова команды** откроется интерактивный интерфейс для тонкой настройки, который позволяет представить древовидную структуру всех опций ядра (драйверы устройств, поддержка файловых систем, сетевые функции и т.д.) и выбирать нужные.

В menuconfig все навигируется стрелками. 

**Выбор опций осуществляется с помощью Enter, включение (Y), выключение (N) осуществляется с помощью соответствующих клавиш. Кроме того может пригодиться поиск нужных опций конфигурации и их зависимостей (/).**

По завершении выбора нужных настроек все сохраняется в файл конфигурации .config.

### Шаг 3. Установка 2-х недостающих параметров

Необходимо добавить в конфигурацию ядра поддержку следующих параметров отвечающих за работу с I2C-устройствами:

- I2C_MV64XXX
- I2C_CHARDEV

### Шаг 4. Сборка ядра

По завершении указанных действий, можно произвести попытку запуска сборки командой.

```
make ARCH=riscv CROSS_COMPILE=riscv64-linux-gnu- -j$(nproc)
```

Ждем... ~10 минут

### Шаг 5. Перенос готового ядра на SD-карту

```
cp arch/riscv/boot/Image /run/media/taranev/BOOT/
```

### Шаг 6. Запуск одноплатника и подключение к консоли

Подготовимся к работе с первичной консолью операционной системы.

```
tio -b 115200 /dev/ttyUSB0
```

**Достанем готовую SD-карту из картридера и вставим в слот на на одноплатнике**

**Подадим питание по Type-C интерфейсу**
